how to build this blog

30 Dec 2012 By amyangfei

记录使用jekyll，bundle搭建这个博客的过程

系统环境Mac OS X 10.8.2，这个博客主要通过jekyll生成静态页面，使用ruby提供的一些扩展插件，比如按月份进行归档等，配置环境的过程如下。

安装ruby，rubygems

sudo port ruby
sudo port install rb-rubygems
sudo gem install rubygems-update
sudo gem update --system

安装bundle

sudo gem install bundler

安装jeklly，rdiscount（支持markdown语法，rdiscount github），liquid（支持jekyll模板渲染，Liquid Extensions）。可以写到一个Gemfile文件中然后 bundle install。

bundle install

Gemfile文件内容

source "http://rubygems.org"

gem "jekyll", "0.10.0"
gem "jekyll_ext"
gem "liquid", "2.2.2"
gem "rdiscount"

关于jekyll的一些扩展插件，可以参考 jekyll plugins 和 rfelix's jelyll extensions 。

生成静态页面，运行服务器

bundle exec ejekyll --server

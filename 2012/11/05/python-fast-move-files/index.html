How to move files fast with python

How to move files fast with python

2012-11-05 By amyangfei

介绍一种利用Python简单快速在磁盘中移动文件的方法。

需求很简单，有一个目录下有很多子目录，每个子目录中都会有一些具有特定格式的文件，比如jpg图片。需要将这些文件移动到一个统一的文件夹中，按照指定格式统一命名。可以使用python提供的os.walk遍历子目录，使用os.rename更改文件属性实现移动。另外关于使用shutil.move，操作因其基于复制效率较低，查看Python的官方文档有这样的说明：

shutil.move(src, dst)
If the destination is a directory or a symlink to a directory, then src is moved inside that directory.
The destination directory must not already exist. If the destination already exists but is not a directory, it may be overwritten depending on os.rename() semantics.
If the destination is on the current filesystem, then os.rename() is used. Otherwise, src is copied (using shutil.copy2()) to dst and then removed.

不多说，直接贴代码：

import os

dst = './dst'
count = 0

def movfiles(pathstr):
    global count
    #os.walk yields a 3-tuple (dirpath, dirnames, filenames)
    for ret in os.walk(pathstr):
        for fname in ret[-1]:
            if len(fname)>0 and '.jpg' in fname:
                shutil.move(os.path.join(ret[0], fname), 
                    os.path.join(dst, str(count)+'_'+fname))
                count += 1

后来使用os.rename和shutil的方法进行了一下对比，移动60张jpg文件共139MB从一个目录到另一个目录，os.rename平均时间是0.004508s，shutil.move的平均时间是0.005764s，耗时比os.rename多27.9%
